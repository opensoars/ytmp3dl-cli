# ytmp3dl-cli

Youtube mp3 downloader - command line interface module.

This module uses the [ytmp3dl-core](https://github.com/opensoars/ytmp3dl-core) module.

---

# Install

(FUTURE) `npm install -g ytmp3dl-cli`

# Requirements

- `Node.js` (>=10.16.0) + `NPM`
- Globally installed `ffmpeg`
